# rent-auto

## Описание
Приложение для аренды авто.

### Используемый стек технологий
- Hibernate
- Spring (Boot, MVC, Data JPA, Security)
- PostgreSQL
- Maven
- JUnit + Mockito

### Запуск
1. docker-compose up
2. http://localhost:5556/login

### Вход
- администратор
  - login: admin
  - password: admin
- клиент
  - login: user
  - password: user
